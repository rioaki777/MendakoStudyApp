# はじめに
これはAndroidアプリのrepositoryです。
4-6才児向けの知育アプリを開発する。主要なキャラクターとしてタコのメンダコが登場する。

# 主な教材
- 足し算(20以下)
  - 受講する度に異なる問題がでる。正答率があがるとより大きな数で問題がでやすくなる。
- 引き算(10以下)
  - 受講する度に異なる問題がでる。正答率があがるとより大きな数で問題がでやすくなる。
- ひらがなの書き方
  - 管理画面にて登録された短い文が問題となる。書き順とおりに文字をなぞることで完了する。

# 機能
- 教材受講
- ポイント管理
  - 教材受講でポイント取得
- ポイント交換
  - 集めたポイントと商品を交換
    - メンダコのご飯
    - メンダコのアクセサリー
    - メンダコの家の家具
- アクセサリーの付け替え
- メンダコへのご飯プレゼント
- メンダコの家の家具配置
