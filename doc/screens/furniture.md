# 家具配置画面

## 画面概要
所持している家具をメンダコの部屋に配置・移動・取り外しする画面。ドラッグ＆ドロップで直感的に操作できる。

## 画面要素

| 要素 | 種類 | 説明 |
|------|------|------|
| 部屋ビュー | Custom View | 家具を自由に配置できるインタラクティブなエリア |
| 配置済み家具 | View（ドラッグ可） | 長押しでドラッグ開始、離すと配置確定 |
| 所持家具パネル | RecyclerView（横スクロール） | 未配置の所持家具一覧。タップで部屋ビューに追加 |
| 「取り除く」エリア | View | ゴミ箱アイコン。ドラッグして離すと配置解除（所持に戻る） |
| 「きめる」ボタン | Button | 現在の配置をDBに保存して前の画面へ戻る |
| 「やめる」ボタン | Button / ActionBar | 変更を保存せず前の画面へ戻る |

## 画面遷移

| 操作 | 遷移先 |
|------|--------|
| 「きめる」タップ | メンダコのお部屋画面 (`mendako_room`) |
| 「やめる」 / バック | メンダコのお部屋画面 (`mendako_room`)（変更破棄） |

**遷移元:** メンダコのお部屋画面 (`mendako_room`)

## 状態管理

| 状態 | 型 | 説明 |
|------|----|------|
| 所持家具一覧 | List\<OwnedItem\> | DB: `OwnedItem` (category=FURNITURE) |
| 配置中の家具（編集中） | List\<FurniturePlacement\> | ローカル一時状態 |
| 保存済み配置 | List\<FurniturePlacement\> | DB: `FurniturePlacement` テーブル |

```kotlin
data class FurniturePlacement(
    val itemId: Int,
    val x: Float,   // 部屋ビュー内の相対座標 (0.0〜1.0)
    val y: Float
)
```

## インタラクション

1. **画面表示**: 所持家具・配置済み家具をDBから取得し表示
2. **所持家具パネルのアイテムタップ**: 部屋ビューの中央付近に追加（ローカル一時状態に追加）
3. **配置済み家具の長押し → ドラッグ**: 指に追従して移動
4. **ドロップ（指を離す）**:
   - 部屋ビュー内: その座標に配置確定（一時状態を更新）
   - 「取り除く」エリア上: 配置を解除し所持家具パネルに戻す
5. **「きめる」タップ**: 一時状態をDBに保存 → 前画面へ戻る
6. **「やめる」**: 一時状態を破棄 → 前画面へ戻る

## 制約

- 同じ家具は1部屋に1つのみ配置可能
- 配置座標は部屋ビューの横×縦に対する相対座標（0.0〜1.0）で保存し、画面サイズ変化に対応する
