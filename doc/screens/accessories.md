# アクセサリー付け替え画面

## 画面概要
所持しているアクセサリーを一覧表示し、メンダコに装着・取り外しを行う画面。変更はリアルタイムにプレビューとして反映される。

## 画面要素

| 要素 | 種類 | 説明 |
|------|------|------|
| メンダコプレビュー | ImageView | 現在選択中のアクセサリーを合成したプレビュー |
| アクセサリー一覧 | RecyclerView | 所持しているアクセサリーカードのリスト |
| 装着中バッジ | View | 現在装着中のアクセサリーに「つけてる」バッジ表示 |
| 「きめる」ボタン | Button | 変更をDBに保存して前の画面へ戻る |
| 「やめる」ボタン | Button / ActionBar | 変更を保存せず前の画面へ戻る |

### アクセサリーカードの要素
- アクセサリー画像（ImageView）
- アクセサリー名（TextView）
- 装着中バッジ（TextView）

## 画面遷移

| 操作 | 遷移先 |
|------|--------|
| 「きめる」タップ | メンダコのお部屋画面 (`mendako_room`) |
| 「やめる」 / バック | メンダコのお部屋画面 (`mendako_room`)（変更破棄） |

**遷移元:** メンダコのお部屋画面 (`mendako_room`)

## 状態管理

| 状態 | 型 | 説明 |
|------|----|------|
| 所持アクセサリー一覧 | List\<OwnedItem\> | DBから取得（category=ACCESSORY） |
| 現在の装着状態（編集中） | MutableSet\<AccessoryId\> | 操作中の一時的な状態 |
| 保存済み装着状態 | Set\<AccessoryId\> | DB: `UserState.equippedAccessories` |

## インタラクション

1. **画面表示**: 所持アクセサリーと現在の装着状態をDBから取得
2. **アクセサリーカードタップ**:
   - 未装着 → 装着（一時状態に追加、プレビュー更新）
   - 装着中 → 取り外し（一時状態から削除、プレビュー更新）
3. **「きめる」タップ**: 一時状態をDBに保存 → 前画面へ戻る
4. **「やめる」**: 一時状態を破棄 → 前画面へ戻る

## 制約

- 装着可能なアクセサリーの部位制限（例: 頭は1つまで）がある場合は将来の拡張として検討
- 現フェーズでは部位制限なし、複数同時装着可能とする
