# ご飯プレゼント画面

## 画面概要
所持しているメンダコのご飯を選んで与える画面。ご飯を与えるとメンダコが喜ぶアニメーションが流れる。ご飯は消耗品のためプレゼントすると所持数が減る。

## 画面要素

| 要素 | 種類 | 説明 |
|------|------|------|
| メンダコアニメーション | ImageView / Lottie | 通常時・食べているとき・喜んでいるときの状態 |
| ご飯一覧 | RecyclerView | 所持しているご飯カードのリスト |
| 所持数バッジ | TextView | 各ご飯カードに個数を表示（例: 「×3」） |
| 「あげる」ボタン | Button | 選択中のご飯を1つプレゼント |
| 「もどる」ボタン | Button / ActionBar | メンダコのお部屋画面へ戻る |

### ご飯カードの要素
- ご飯画像（ImageView）
- ご飯名（TextView）
- 所持数（TextView）: 0の場合はグレーアウト

## 画面遷移

| 操作 | 遷移先 |
|------|--------|
| バック / 「もどる」 | メンダコのお部屋画面 (`mendako_room`) |

**遷移元:** メンダコのお部屋画面 (`mendako_room`)

## 状態管理

| 状態 | 型 | 取得元 |
|------|----|--------|
| 所持ご飯一覧 | List\<OwnedFoodItem\> | Room DB: `OwnedItem` (category=FOOD) |
| 選択中のご飯 | FoodItemId? | ローカル（画面内の選択状態） |
| メンダコのアニメーション状態 | FoodState | IDLE / EATING / HAPPY |

```kotlin
data class OwnedFoodItem(
    val itemId: Int,
    val name: String,
    val imageRes: Int,
    val quantity: Int
)
```

## インタラクション

1. **画面表示**: 所持ご飯一覧をDBから取得して表示
2. **ご飯カードタップ**: 選択状態にする（ハイライト）
3. **「あげる」タップ**:
   - 選択なし: 「ごはんをえらんでね」Toast
   - 所持数0: ボタン無効（グレーアウト）
   - 所持数1以上:
     1. メンダコのアニメーションを EATING に切り替え
     2. `OwnedItem.quantity` を -1 更新
     3. アニメーション完了後 HAPPY に切り替え → 数秒後 IDLE に戻る
     4. 所持数の表示を更新
4. **所持数が0になったご飯**: カードをグレーアウト表示、「あげる」ボタンを無効化
