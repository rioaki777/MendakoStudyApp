# 教材選択画面

## 画面概要
受講する教材（足し算・引き算・ひらがな）を選択する画面。各教材の現在の正答率を表示し、子どもが自分の得意・苦手を意識できるようにする。

## 画面要素

| 要素 | 種類 | 説明 |
|------|------|------|
| 「もどる」ボタン | Button / ActionBar | ホーム画面へ戻る |
| 足し算カード | CardView | アイコン＋「たしざん」テキスト＋現在の正答率 |
| 引き算カード | CardView | アイコン＋「ひきざん」テキスト＋現在の正答率 |
| ひらがなカード | CardView | アイコン＋「ひらがな」テキスト＋現在の正答率 |

**正答率表示例:** 「せいかいりつ: 80%」（直近20問ベース）

## 画面遷移

| 操作 | 遷移先 |
|------|--------|
| 足し算カードタップ | 足し算レッスン画面 (`addition_lesson`) |
| 引き算カードタップ | 引き算レッスン画面 (`subtraction_lesson`) |
| ひらがなカードタップ | ひらがな書き方レッスン画面 (`hiragana_lesson`) |
| バック / 「もどる」 | ホーム画面 (`home`) |

## 状態管理

| 状態 | 型 | 取得元 |
|------|----|--------|
| 足し算正答率 | Float (0.0〜1.0) | Room DB: `LessonStats` (subject=ADDITION) |
| 引き算正答率 | Float | Room DB: `LessonStats` (subject=SUBTRACTION) |
| ひらがな正答率 | Float | Room DB: `LessonStats` (subject=HIRAGANA) |

## インタラクション

- **画面表示時**: 各教材の正答率をDBから取得して表示
- **カードタップ**: 選択した教材のレッスン画面へ遷移（選択した教材IDをIntentで渡す）
