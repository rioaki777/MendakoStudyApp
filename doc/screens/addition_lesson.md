# 足し算レッスン画面

## 画面概要
足し算の問題を1問ずつ出題する画面。20以下の足し算を扱い、正答率に応じて出題する数の大きさが変化する（適応難易度）。4〜6歳向けのため、問題は視覚的（りんごの絵など）に表示することを推奨する。

## 画面要素

| 要素 | 種類 | 説明 |
|------|------|------|
| 問題文 | TextView | 例: 「3 + 5 = ?」大きなフォントで表示 |
| ビジュアル補助 | ImageView × N | 数字に対応する絵（くだもの等）を並べて表示（任意） |
| 選択肢ボタン × 4 | Button | 正解1つ＋ダミー3つをシャッフルして表示 |
| 問題番号 | TextView | 「1 / 5」形式（1セット5問） |
| 「やめる」ボタン | Button | 確認ダイアログを挟んでホーム画面へ戻る |

## 画面遷移

| 操作 | 遷移先 |
|------|--------|
| 5問終了 | レッスン結果画面 (`lesson_result`) |
| 「やめる」→確認OK | ホーム画面 (`home`) |

**遷移元:** 教材選択画面 (`lesson_selection`)

## 状態管理

| 状態 | 型 | 説明 |
|------|----|------|
| 現在の問題インデックス | Int | 0〜4 |
| 問題リスト | List\<AdditionQuestion\> | セッション開始時に一括生成（適応難易度を参照） |
| 回答済み結果 | List\<Boolean\> | 各問正解/不正解 |

```kotlin
data class AdditionQuestion(
    val operandA: Int,  // 1〜上限値
    val operandB: Int,  // 1〜(上限値 - operandA)
    val answer: Int,    // operandA + operandB
    val choices: List<Int>  // 4択（answerを含む）
)
```

## インタラクション

1. **画面表示**: 問題リストを生成（詳細は `features/adaptive_difficulty.md`）し、最初の問題を表示
2. **選択肢タップ**:
   - 正解の場合: 選択肢を緑にハイライト → 少し待って次の問題へ
   - 不正解の場合: 選択肢を赤にハイライト → 正解を緑に表示 → 少し待って次の問題へ
3. **5問完了**: レッスン結果画面へ遷移（正解数・獲得ポイントを渡す）
4. **「やめる」タップ**: 「やめますか？」確認ダイアログを表示

## 出題ルール

- 1セット5問を一括生成してから出題する
- 答えが1〜20の範囲に収まる組み合わせのみ
- 同じ問題が連続して出ないよう制御
- 選択肢の範囲: 正解 ±1〜5 の整数4つ（重複・負数・20超えを除く）
- 詳細なアルゴリズムは `features/adaptive_difficulty.md` を参照
