# レッスン結果画面

## 画面概要
レッスン1セット終了後に表示される結果画面。正解数・獲得ポイントを表示し、子どもの達成感を高める演出（メンダコのリアクション）を行う。

## 画面要素

| 要素 | 種類 | 説明 |
|------|------|------|
| メンダコのリアクション画像 | ImageView | 正答率に応じて「よくできました」「もう少し」などの表情 |
| 結果テキスト | TextView | 例: 「5もん中 4もん せいかい！」 |
| 獲得ポイント表示 | TextView | 例: 「＋40ポイント ゲット！」 |
| 累計ポイント表示 | TextView | 例: 「ごうけい: 123ポイント」 |
| 「もう一度やる」ボタン | Button | 同じ教材のレッスンを再開 |
| 「ほかをやる」ボタン | Button | 教材選択画面へ戻る |
| 「おうちへかえる」ボタン | Button | ホーム画面へ戻る |

## 画面遷移

| 操作 | 遷移先 |
|------|--------|
| 「もう一度やる」 | 直前と同じレッスン画面（新しい問題セット） |
| 「ほかをやる」 | 教材選択画面 (`lesson_selection`) |
| 「おうちへかえる」 | ホーム画面 (`home`) |

**遷移元:** 足し算 / 引き算 / ひらがなレッスン画面

## 状態管理（Intentで受け取るデータ）

| パラメータ | 型 | 説明 |
|-----------|-----|------|
| `subjectType` | SubjectType | ADDITION / SUBTRACTION / HIRAGANA |
| `correctCount` | Int | 正解数 |
| `totalCount` | Int | 出題数（通常5） |
| `earnedPoints` | Int | このセッションで獲得したポイント |

## インタラクション

1. **画面表示**: Intentのパラメータをもとに結果を表示
2. **ポイント加算**: `earnedPoints` をDBへ書き込み（画面表示と同時に実行）
3. **正答率の更新**: `LessonStats` を更新（適応難易度に反映）
4. **リアクション画像の決定**:
   - 正答率 80%以上: 「よくできました」
   - 正答率 40〜79%: 「いいちょうし」
   - 正答率 40%未満: 「もう少しがんばろう」

## ポイント計算ルール

詳細は `features/point_system.md` を参照。
- 基本: 1正解 = 10ポイント
- ボーナス: 全問正解 +20ポイント
