# ホーム画面

## 画面概要
アプリ起動後に最初に表示されるメイン画面。メンダコのキャラクターを表示しながら、各機能への入口となる。子ども（4〜6歳）が直感的に操作できるよう、大きなボタンとアイコンで構成する。

## 画面要素

| 要素 | 種類 | 説明 |
|------|------|------|
| メンダコ画像 | ImageView | 現在のアクセサリーを装着した状態のメンダコを表示 |
| 現在のポイント表示 | TextView | 「⭐ 123pt」形式で常時表示 |
| 「べんきょうする」ボタン | Button | 教材選択画面へ遷移 |
| 「おみせ」ボタン | Button | ショップ（ポイント交換）画面へ遷移 |
| 「メンダコのへや」ボタン | Button | メンダコのお部屋画面へ遷移 |

## 画面遷移

| 操作 | 遷移先 |
|------|--------|
| 「べんきょうする」タップ | 教材選択画面 (`lesson_selection`) |
| 「おみせ」タップ | ショップ画面 (`shop`) |
| 「メンダコのへや」タップ | メンダコのお部屋画面 (`mendako_room`) |

**遷移元:** アプリ起動（ランチャー）

## 状態管理

| 状態 | 型 | 取得元 |
|------|----|--------|
| 現在ポイント | Int | Room DB: `UserState.currentPoints` |
| メンダコの装着アクセサリー | List\<AccessoryId\> | Room DB: `UserState.equippedAccessories` |

## インタラクション

- **アプリ起動時**: DBからユーザー状態を読み込み、メンダコ画像とポイントを更新する
- **ボタンタップ**: 対応する画面へ遷移（バック操作でホームへ戻れる）
- **メンダコ画像タップ**: 特に機能なし（または簡単なアニメーション再生）
